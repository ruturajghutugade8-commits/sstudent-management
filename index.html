Student Management System

This is a Java-based project that demonstrates student record management using MySQL and JDBC.

Project Files

  • DBDemo.java → Source code
  • x.sql.gz → Database schema
  • makefile → Compile/run instructions
  • mysql-connector-java-5.1.34-bin.jar → JDBC driver

Features

  • Add new students
  • Update student details
  • Delete student records
  • View student list

⚡ Note: This project runs with Java + MySQL on your computer. GitHub Pages can only show this description, not run Java code.
